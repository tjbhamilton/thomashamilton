# Reverb Consulting Website

## Starts Gulp Build

```bash
nvm use 5.3 # skip if installed already
npm start
```

## Jekyll

```bash
bundle exec jekyll serve
```
to build this one time without starting a server run

```bash
bundle exec jekyll build
```

```bash
jekyll exec serve --drafts
```

to work on the blog


## Deployment

```bash
gulp
(ctrl-c)
bash deploy.sh
```
